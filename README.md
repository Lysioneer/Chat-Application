# Chat-Application

I'm making this project just to practise my coding skills.

It is supposed to be a Chat application, where a user is able to register with use of username,
password and email.

He will then activate the account via email that is sent to him from the registration.

Then onwards with each login(username and password) he is gonna be remembered with help of JWT.

After logging in he/she is gonna see list of chats he is in, either as a member or owner.

He is gonna be able to tell if there are any new messages in any chat.

He will be able to create a new chat, select members, and send messages in the chats.

Every message in chat will have: author, date, text itself, avatar of user.

Every chat will have an avatar and a name.
---------------------------------------------------------------------------------------------------
For each input there will be a "check" of the input if it's in a correct form,
on BE as well as on FE side.

There is one Main branch with working code.

Each additional branch is supposed to have 1 new feature in them, which are then merged into Main.
